# spring_base

## 开发指南

### 技术栈

- Java 17
- Spring Boot 3.1.2
- Gradle 8.2.1

### 本地依赖

#### Docker desktop only

### 本地构建

```
./gradlew clean build 
```

### 本地运行

```
./gradlew bootRun
```

### Swagger

Local environment: http://localhost:9090/swagger-ui.html

### 格式代码

```
./gradlew spotlessApply
```
